# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. The full text is available at the link above and is
incorporated here by reference.

In short: contributors and maintainers pledge to make participation in this
project a respectful, constructive experience for everyone, regardless of
background or identity. Unacceptable behavior is defined in the Covenant.

## Scope

This Code of Conduct applies within all project spaces, including the
repository, its issues, pull requests, discussions, and any other
communication channels tied to the project, and when an individual is
representing the project in public.

## Reporting

Reports go to **conduct@framedrag.dev** (to be confirmed before the first
public release). Reports are handled privately. Project maintainers are
obligated to respect the privacy and security of the reporter.

## Enforcement

Maintainers will review each report and respond as outlined in the
Covenant's enforcement guidelines. Consequences range from a private warning
to a permanent ban, depending on severity and history.
