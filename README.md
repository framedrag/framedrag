# framedrag

**Curated IP reputation feeds, dragged into the null route.**

The IP-blocking half of pfBlockerNG, decoupled from pfSense. framedrag
consumes curated IP reputation feeds and turns them into live firewall
policy, with silent-failure detection as the core feature rather than an
afterthought.

Under construction. See `docs/SPEC.md` for the full design.
